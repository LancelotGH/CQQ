# Game Asset Specifications

Here are the specifications for the image assets required for the game. All images should be in **PNG format** with transparency.

## 1. Grid & UI
- **Grid Tile**: 70x70 px
- **Icons (Sword, Bow, Staff)**: 30x30 px or 40x40 px

## 2. Units (Player & Enemy)
All standard units should fit within the grid cell.
- **Size**: 60x60 px (Recommended)
- **Max Size**: 64x64 px (To leave small padding in 70px cell)

| Unit Type | Description | Color Code (Current) |
| :--- | :--- | :--- |
| **Warrior** | Melee, Sword | Blue |
| **Archer** | Ranged, Bow | Green |
| **Mage** | AoE, Staff | Purple |
| **Goblin (Melee)** | Enemy Melee | Red |
| **Spear (Ranged)** | Enemy Ranged | Orange |
| **Warlock (AoE)** | Enemy Area Dmg | Blue-Grey |

## 3. Bosses
Bosses occupy a 2x2 grid space.
- **Size**: 130x130 px (Recommended)
- **Max Size**: 140x140 px

## 4. UI Buttons
- **Restart / Exit Buttons**: 50x50 px
